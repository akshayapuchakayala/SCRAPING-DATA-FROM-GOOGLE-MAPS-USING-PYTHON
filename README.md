# SCRAPING-DATA-FROM-GOOGLE-MAPS-USING-PYTHON

## 📌 Project Overview

This project focuses on **extracting business-related data from Google Maps** using web scraping techniques.
The collected data can be used for **market analysis, lead generation, research, and location-based insights**.

The system automates the process of gathering structured information such as business names, addresses, ratings, and contact details from Google Maps search results.



## 🎯 Objectives

* Automate data collection from Google Maps
* Reduce manual effort in gathering business information
* Store scraped data in a structured format for analysis
* Enable location-based business insights



## 🛠️ Technologies Used

* **Python**
* **Selenium WebDriver**
* **BeautifulSoup**
* **Pandas**
* **Google Chrome & ChromeDriver**



## 📂 Features

* Search businesses by keyword and location
* Extract business details:

  * Business Name
  * Address
  * Phone Number (if available)
  * Ratings & Reviews
  * Website link
* Export data into **CSV format**
* Scalable for different locations and business categories


## ⚙️ System Workflow

1. User provides search keyword and location
2. Selenium opens Google Maps automatically
3. Business listings are loaded dynamically
4. Required data fields are extracted
5. Data is cleaned and stored in CSV format


## 🚀 Applications

* Digital marketing & lead generation
* Business intelligence
* Market research
* Location-based service analysis
* Academic and learning purposes


## ⚠️ Limitations

* Google Maps frequently updates its UI
* Excessive scraping may lead to temporary blocking
* Requires stable internet and browser compatibility
